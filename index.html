Whats Scroll Snap?

Scroll snap allows content to automatically snap to a specific point on the screen when the user finishes scrolling, creating a more fluid and controlled browsing experience.
It serves to create more interactive and visually appealing browsing experiences, especially on websites with multiple sections or slides.

Code Explanation

This CSS uses scroll-snap-type to control scrolling behavior. The y mandatory setting ensures content snaps vertically along the Y-axis.
scroll-padding-top adds top padding to account for fixed elements like a navigation bar. scroll-snap-align and scroll-snap-stop are applied to sections to align them correctly when scrolling.
Each section is set to 100vh, making it occupy the full viewport height.

overflow: hidden prevents content overflow, while z-index manages element stacking, and padding adds internal spacing.
